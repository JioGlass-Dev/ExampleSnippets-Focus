# Focus
Usage Exapmle of OnFocusEnter and OnFocusExit

## Scripts 

### `FocusManager.cs`
Shows example of accessing callbacks called during focus enter and focus exit.</br>
- Interfaces to implement focus funtionality
```cs
public class FocusManager : MonoBehaviour, IFocusable
```
- Methods to implement focus funtionality
```cs
public void OnFocusEnter()
{ }
public void OnFocusEnter()
{ }
```

## How to use?
1. Download and unzip this project.
2. Open the project using Unity Hub.
3. Download and import the latest version of JMRSDK package.
4. Open and play the Focus scene from Assets/Scenes folder.
